**Full-Stack Developer | DevOps | Problem Solver a 360°**

Mi occupo di digitalizzare e ottimizzare i processi aziendali partendo dalle fondamenta fisiche fino ad arrivare al cloud. Non mi limito a scrivere codice: metto le mani sull'elettronica, configuro print server hardware e sviluppo piattaforme SaaS scalabili. Costruisco soluzioni complete a 360 gradi.

### 🚀 Cosa faccio
- 💻 **Sviluppo Web & SaaS:** Architetture moderne, PWA e piattaforme multi-tenant.
- ⚙️ **DevOps, Home Lab & Self-Hosting:** Containerizzazione, gestione di server fisici con Proxmox, VPS e automazione processi aziendali.
- 🔌 **Hardware & Elettronica:** Soluzioni ibride, configurazione di infrastrutture locali, Raspberry Pi e print server.
- 🤖 **AI & Local LLM:** Esecuzione di modelli linguistici locali (tramite Ollama) e integrazione di sub-agenti AI per ottimizzare i flussi di sviluppo.
- 💰 **Sostenibilità economica:** Integro soluzioni open-source per contenere i costi architetturali e di gestione.
